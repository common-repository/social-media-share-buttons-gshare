= How to Install Gshare? =

1. Go to your Dashboard
2. Opt-in to Plugins > Add New 
3. Search for Gshare
4. Click install and Activate

= Configuration =

1. Go to Gshare Dashboard
2. Customize Your Gshare Buttons for global settings
3. Choose from over 35 social media share buttons
4. Set from up to 4 types of button positions
5. Choose the display where they will appear. You can choose both Pages and Post/ any one/ None
6. You can also add and modify social icons block on gutenberg editor individually by diabling global settings.
7. Simply find or search 'Social Media Share Button - Gshare' block listed with other blocks and click to add.
